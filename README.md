# AsunaTracks Sync for Seanime has moved

This old standalone repository is no longer the active source for the Seanime extension.

Use the Seanime-Stuff repository instead:

```text
https://github.com/Kolex06/Seanime-Stuff
```

Install the extension in Seanime with this manifest URL:

```text
https://raw.githubusercontent.com/Kolex06/Seanime-Stuff/refs/heads/main/plugins/asunatracks-sync.json
```

The current provider, marketplace entry, icon, README, and version updates are maintained there now.
